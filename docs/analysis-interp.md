# interp — NEB 路径重新插值

从一条 NEB 路径（若干 POSCAR 文件）出发，通过描述语句生成新路径。支持三次样条插值和逐对线性插值，自动处理分数坐标的周期性边界条件。

## 用法

```bash
# 使用路径描述语句
vaspauto analysis interp --spec "[0,1,3:2]" -p POSCAR_00 POSCAR_01 POSCAR_02 POSCAR_03 -d output/

# 传统写法：相邻结构逐对插值
vaspauto analysis interp -p POSCAR_00 POSCAR_01 -n 5 -d output/
```

## 选项

| 参数 | 说明 |
|------|------|
| `--spec` | 路径描述语句，见下文语法 |
| `-p, --path` | 旧路径的 POSCAR 文件列表 |
| `-i, --init` / `-f, --final` | 仅指定初末态（等价于 `-p init final`） |
| `-n, --num` | 插值点数（`--spec` 时忽略） |
| `-d, --dir` | 输出目录 |
| `--fix` | 使用 Wigner-Seitz 最近镜像修正 |
| `--old-fix` | 使用旧版 ±1 修正 |
| `--no-startpoint` / `--no-endpoint` | 排除首/尾端点 |
| `--start-idx` | 输出文件起始编号，默认 0 |
| `--prec` | 输出精度（小数位数），默认 10 |

输出格式为 NEB 标准目录结构：`00/POSCAR`、`01/POSCAR`、...

---

## 路径描述语句语法

语句由路径段构成，段之间用逗号 `,` 拼接。

### 单个结构

| 语法 | 含义 |
|------|------|
| `A` | 旧路径上序号为 *A* 的结构 |

### 插值表达式

```
[锚点:插值方式]
```

**锚点**：用 `-` 连接序号表示连续区间，用 `,` 列出离散序号，两者可混合：

| 写法 | 选取的结构 |
|------|-----------|
| `1-3` | 1, 2, 3 |
| `1,2,4` | 1, 2, 4 |
| `1-2,4` | 1, 2, 4 |

**插值方式**：

| 语法 | 含义 |
|------|------|
| `:n` | 沿曲线均匀生成 *n* 个插值点。两点退化为线性，两点以上使用三次样条。不保证经过中间锚点 |
| `::n` | 逐对插值：对每一对相邻锚点分别生成 *n* 个插值点。保证经过每个锚点结构 |

**开闭**：方括号决定插值后输出是否包含两端点：

| 语法 | 输出 |
|------|------|
| `[...]` | 包含两端点 |
| `[...)` | 不含右端点 |
| `(...]` | 不含左端点 |

闭区间 `[...]` 可省略方括号，直接写 `锚点:n`。

### 示例

```
# 闭区间：0-4 五个结构，插值 5 个点，输出含首尾
0-4:5

# 半开：输出不含末尾
[0-4:5)

# 离散锚点：选取 1,2,4，插值 5 个点
[1,2,4:5]

# 逐对插值：经过 0,1,2,3 每个结构
0-3::2

# 拼接：结构 0，再 0 到 2 插值 4 个点，最后结构 3
0,0-2:4,3
```

## 注意事项

- 插值前确保对应原子排序一致，否则路径无物理意义
- `1-3:5` 不保证经过结构 2；如需保证经过每个结构，用 `::` 逐对插值
