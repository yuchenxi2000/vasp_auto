# 路径重新插值方法

## 旧路径插值生成的新路径的表示格式

单个点选取：

1. $A$ 表示旧路径上序号为 $A$ 的单个点

2. $A_1-A_n$ 表示旧路径上序号从 $A_1$ 到 $A_n$ 的 $n$ 个点

以下是插值生成点的格式， $m$ 表示插值点个数，不包括端点

1. $A_1,A_n$ 间连线均匀插值 $A_1-A_n:m$

2. 连线均匀插值简化版：对 $A_1,A_2,\dots,A_n$ 所有相邻点两两连线进行插值 $A_1-A_n::m$

3. $A_1,A_2,\dots,A_n$ 间连弧均匀插值 $A_1~A_n:m$

所有这些新路径上的点可以通过逗号相连接组成更长的路径

## 插值算法限制

必须保证两个被插值结构之间的原子排序合理，也就是对应原子距离较近，否则不保证路径正确性。

插值算法会自动考虑周期性边界条件带来的原子分数坐标的不确定性，但需要手动调整原子排序使两个结构对齐。

## 原子分数坐标的不确定性的解决方法

1. 手动调整分数坐标使得初末态对齐

2. 旧算法，简单算法：先将所有分数坐标归一化到 $[0,1)$ 区间内。如果初末态分数坐标相差大于 $0.5$ ，那么将其中一个加一或者减一，使得坐标最近。

3. 基于Wigner-Sitz原胞的算法：求两个原子所有周期性镜像里面笛卡尔距离最近的一对镜像，它们的坐标就是使得初末态结构对齐的坐标
