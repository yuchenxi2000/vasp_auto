# 路径重新插值方法

通过描述语句从一条旧路径生成新路径。语句由路径段构成，段之间用逗号 `,` 拼接。

---

## 路径段语法

### 单个结构

| 语法 | 含义 |
|------|------|
| `A` | 旧路径上序号为 *A* 的结构 |

### 插值表达式

选取旧路径上若干结构作为锚点，沿其构成的曲线生成插值点。

```
[锚点:插值方式]
```

**锚点**：用 `-` 连接序号表示连续区间，用 `,` 列出离散序号，两者可混合：

| 写法 | 选取的结构 |
|------|-----------|
| `1-3` | 1, 2, 3 |
| `1,2,4` | 1, 2, 4 |
| `1-2,4` | 1, 2, 4 |

**插值方式**：

| 语法 | 含义 |
|------|------|
| `:n` | 沿曲线均匀生成 *n* 个插值点。两点退化为线性，两点以上使用三次样条插值。不保证经过中间锚点 |
| `::n` | 逐对插值：对每一对相邻锚点分别生成 *n* 个插值点。保证经过每个锚点结构 |

**开闭**：方括号决定插值后输出是否包含两端点：

| 语法 | 输出 |
|------|------|
| `[...]` | 包含两端点 |
| `[...)` | 不含右端点 |
| `(...]` | 不含左端点 |

闭区间 `[...]` 可省略方括号，直接写 `锚点:n`。

---

## 示例

```
# 闭区间：0-4 五个结构，插值 5 个点，输出含首尾
0-4:5

# 半开：输出不含末尾（4 不在输出中）
[0-4:5)

# 离散锚点：选取 1,2,4，插值 5 个点
[1,2,4:5]

# 离散锚点半开：不含末尾
[1,2,4:5)

# 逐对插值：经过 0,1,2,3 每个结构
0-3::2

# 混合：锚点为 0,1,2,4，插值 5 个点不含末尾
[0-2,4:5)

# 拼接：结构 0，再 0 到 2 插值 4 个点，最后结构 3
0,0-2:4,3
```

---

## 注意事项

- 插值前确保对应原子排序一致，否则路径无物理意义
- 插值算法自动处理分数坐标的周期性边界条件（选取最近镜像）
- `1-3:5` 不保证经过结构 2；如需保证经过每个结构，用 `::` 逐对插值
